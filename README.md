# 🏠 Advanced House Price Predictor

## 📌 Overview
This project predicts house prices using Machine Learning.

## 🚀 Features
- Random Forest Regression
- Predict price based on:
  - Area (sqft)
  - Bedrooms
  - Bathrooms
  - Floors
  - Location (lat, long)
- Interactive Streamlit UI

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## ▶️ Run
```bash
streamlit run app.py
